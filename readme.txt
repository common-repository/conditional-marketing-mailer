Stop Cart Abandonments and Recover Your Lost Revenue, This plugin let you create marketing emails to be sent based on custom conditions

== Description ==
**Stop Cart Abandonments and Recover Your Lost Revenue.**
Some users who go to the checkout page, do not complete their purchase. Our plugin will solve this problem :)

All notices can be created and active on your website. Each notice can have a different set of conditions that need to be met for the notice to be sent.
Notices can be in published or draft status. Only published notices will be active on the site.

= All features =
* For each newsletter, see which subscribers opened it and which links got the most clicks
* Build and send newsletters with WordPress
* Create automatic emails to send notifications
* Send automated welcome emails
* Increase your sales with our emails for WooCommerce
* Configurable messages using shortcodes
* Easy to use newsletter builder
* No configuration needed, works out of the box
* GDPR compliant
